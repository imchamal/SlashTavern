# SlashTavern

SillyTavern 채팅 편의 기능 확장입니다.

## 통합 명령어

`/st` 뒤에 원하는 기능 이름을 입력해서 SlashTavern 기능을 실행할 수 있습니다.

| 명령어 | 기능 |
|---|---|
| `/st search` | 검색 패널 열기 |
| `/st search 키워드` | 채팅에서 키워드 검색 |
| `/st search 원본/바꿀말` | 찾아바꾸기 |
| `/st goto 5` | 5번 메시지로 이동 |
| `/st up` | 채팅 맨 위로 이동 |
| `/st down` | 채팅 맨 아래로 이동 |
| `/st prev` | 현재 위치 기준 이전 메시지로 이동 |
| `/st next` | 현재 위치 기준 다음 메시지로 이동 |
| `/st message-mb` | STMemoryBooks 메모리 경계선으로 이동 |
| `/st clip` | 전체 메시지 복사 |
| `/st clip 2-5` | 2번부터 5번 메시지 복사 |
| `/st word` | 전체 글자수/단어수 확인 |
| `/st word 2-5` | 2번부터 5번 메시지 글자수/단어수 확인 |
| `/st settings` | 설정 패널 열기 |
| `/st messages` | 전체 메시지 관리 패널 열기 |

# SlashTavern
## SlashTavern 통합 명령어

기존 명령어 앞에 `/st`를 붙여서 실행할 수 있습니다.

| 기존 명령어 | `/st` 방식 |
|---|---|
| `/search` | `/st search` |
| `/goto 5` | `/st goto 5` |
| `/up` | `/st up` |
| `/down` | `/st down` |
| `/message-mb` | `/st message-mb` |
| `/collapse 2-5` | `/st collapse 2-5` |
| `/expand 2-5` | `/st expand 2-5` |
| `/clip 2-5` | `/st clip 2-5` |
| `/word` | `/st word` |
| `/edit-mode` | `/st edit-mode` |
| `/messages` | `/st messages` |
